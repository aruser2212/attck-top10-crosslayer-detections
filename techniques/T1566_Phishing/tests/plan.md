# План тестирования: Phishing (T1566)

Цели:
- Поймать Office/Outlook → Script/LOLBIN.
- Поймать HTA/HTML загрузчики (сеть + хост).
- Статические признаки макросов (YARA).

Шаги (лабораторно/описательно):
1) Сгенерировать документ Office с макросом AutoOpen, выполняющим Shell/WScript (на тестовой машине) — ожидаем Sigma #001 и (при скане) YARA.
2) Выполнить mshta с URL/HTA (стенд) — ожидаем Sigma #002 и сетевую сигнатуру HTA.
3) Загрузить .docm/.xlsm через HTTP (стенд/синтетика) — ожидаем сетевую сигнатуру macro-doc.
4) Отметить легит-источники (надстройки, внутренние порталы) → добавить allow-list / тюнинг в DetCards.
